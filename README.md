# diamond-square-fragmentation
This is the fragmentation algorithm used for
ICFHR 2020 Competition on Image Retrieval for Historical Handwritten Fragments

## Requirements
It works with Python3 and the following libraries:
future==0.18.2
numpy==1.19.0
Pillow==7.2.0
pkg-resources==0.0.0
scipy==1.1.0
torch==1.5.1
torchvision==0.6.1
